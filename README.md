# pmm-config

Kometa (formerly Plex Meta Manager) configuration files for automated Plex library management.

## Structure

- `config-clean.yml` — Main Kometa config
- `movies-best.yml` — Best-of movie collections
- `movies-collections.yml` — Curated movie collections
- `movies-general.yml` — General movie metadata
- `movies-holidays.yml` — Holiday-themed collections
- `movies-mcu.yml` — Marvel Cinematic Universe collections
- `movies-special.yml` — Special/themed movie collections
- `networks.yml` — Network-based TV collections
- `tv.yml` — TV show collections
- `tv-meta.yml` — TV show metadata configs
- `tv-networks.yml` — TV network collections
- `aired-refresh.yml` — Hidden collections for recently-aired metadata refresh
- `unmatched-refresh.yml` — Hidden collection to retry matching unmatched items
- `video-overlays.yml` — Video overlay definitions
- `Overlays/` — Custom overlay images
- `Posters/` — Custom collection posters

## Usage

These configs are designed for use with [Kometa](https://github.com/Kometa-Team/Kometa). Point your Kometa config to this directory or copy individual files as needed.
